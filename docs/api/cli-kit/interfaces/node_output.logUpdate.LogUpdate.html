Interface LogUpdate

Hierarchy

  • LogUpdate
  • LogUpdate(...text: string[]): void

  • Log to stdout by overwriting the previous output in the terminal.

    Example

    import logUpdate = require('log-update');

    const frames = ['-', '\\', '|', '/'];
    let i = 0;

    setInterval(() => {
        const frame = frames[i = ++i % frames.length];

        logUpdate(
    `
            ♥♥
    ${frame} unicorns ${frame}
            ♥♥
    `
        );
    }, 80);
